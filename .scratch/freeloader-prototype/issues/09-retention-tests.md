# Retention tests at section boundaries

Type: task (AFK)
Status: open
Blocked by: 05, 07

## Question

Implement the retention loop: detect the WPM cursor crossing a section/chapter boundary, present a skippable modal asking the user to explain the passage's key idea in their own words, send the answer to Claude for grading against the wiki (encouraging, ADHD-friendly feedback), and store results per section so re-tests and progress are possible. Tone and prompt wording matter — keep them kind.
